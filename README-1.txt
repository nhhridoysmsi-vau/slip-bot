=== স্লিপ সার্চ Telegram Bot ===

DEPLOY করার ধাপ:

১. render.com এ যান (সম্পূর্ণ বিনামূল্যে)
২. GitHub account দিয়ে Sign up করুন
৩. New > Web Service > "Deploy from GitHub"
৪. একটা নতুন GitHub repo বানান এই ৩টা ফাইল দিয়ে:
   - bot.py
   - requirements.txt  
   - render.yaml
৫. Deploy হলে আপনার URL পাবেন (যেমন: https://slip-search-bot.onrender.com)
৬. এই URL এ যান: https://slip-search-bot.onrender.com/set_webhook?url=https://slip-search-bot.onrender.com
৭. Done! Telegram Bot কাজ শুরু করবে।

Bot ব্যবহার:
- ছবি পাঠান → সেভ হবে
- নাম লিখুন → খুঁজে দেবে
